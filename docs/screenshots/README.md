## Screenshots Guide

Place your project screenshots in this folder with these exact filenames:

- `dashboard-overview.png`
- `create-thumbnail-editor.png`
- `rater-score.png`
- `ab-test-result.png`
- `ai-creator-output.png`

Tips:
- Use 16:9 images when possible for visual consistency.
- Prefer clear UI states (real outputs, not empty forms).
- Avoid including secrets or personal data in screenshots.
